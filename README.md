# Coco Şol — Café Website (Konzept)


** Demo:** https://blucaryo.github.io/cocosenl/

## Highlights
- 🎨 **Farbig & editorial** — Creme, Vermillion, Senf, Tannengrün; große Serifen-Typografie
- 🍽️ **Speisekarte im Grill-Royal-Stil** — selbstbewusst gesetzt (Name · Hinweis · Preis)
- 🌐 **Sprachumschalter DE / EN** — clientseitig, ohne Reload
- ✨ **Dynamik** — Lade-Screen, Lauftext-Ticker, sanfte Scroll-Einblendungen, Sticky-Header
- 📍 **Google-Maps-Verlinkung** der Adresse (Hero, Besuch, Footer)
- 📱 **Voll responsiv** — Mobile-first, skaliert sauber bis Desktop
- ♿ **Zugänglich** — Tastaturbedienung, Fokus-Stile, `prefers-reduced-motion` respektiert
- ⚡ **Leicht & ohne Tracking** — statisches HTML, Vanilla-JavaScript, keine Frameworks

## Tech-Stack
HTML5 · CSS3 (Custom Properties, Grid/Flexbox) · Vanilla JavaScript (IntersectionObserver)

## Dateien
- `index.html` — Inhalt & Struktur
- `styles.css` — Design, Farben, Animationen
- `app.js` — Lade-Screen, Scroll-Reveals, Sprachumschalter, Sticky-Header

## Vor dem Live-Gang noch einsetzen
